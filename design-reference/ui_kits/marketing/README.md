# Marketing — Trang chủ công khai

Route `/`. Trang giới thiệu công khai của Blackcrest.

- **Landing.jsx** — toàn bộ trang: Header (sticky, blur), Hero (serif display + xem trước cổng PDF), dải số liệu tin cậy, 3 tính năng, CTA nền tối, Footer.
- Sử dụng `Button`, `Badge` từ design system; icon Lucide qua CDN.
- Tông: sáng, nhiều khoảng trắng, viền hairline, một sắc accent violet. Tiêu đề lớn dùng Source Serif 4.

Mở `index.html` để xem.
