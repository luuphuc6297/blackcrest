Routes `/login`, `/register`.

- **Auth.jsx** — bố cục hai cột: form bên trái, panel thương hiệu nền tối (`bg-marketing`) bên phải với trích dẫn serif. Chuyển qua lại giữa Đăng nhập ↔ Yêu cầu truy cập, có màn hình xác nhận.
- Đăng ký là "yêu cầu truy cập" — tài khoản cần quản trị viên xét duyệt (phù hợp mô hình role-based).
