Surface container — hairline border + soft shadow, the Blackcrest depth recipe.

```jsx
<Card title="Hiệu suất danh mục" subtitle="Quý III · 2026" action={<Badge tone="success">+8,42%</Badge>}>
  <Chart />
</Card>

<Card interactive onClick={open}>
  <DocumentRow />
</Card>
```

- **interactive** adds a hover lift. Use **title/subtitle/action** for the standard header, or just drop children for a bare surface.
